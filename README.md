## Installation

1. **Clone the Repository**

```bash
git clone https://github.com/your-username/gnits-chatbot.git
cd gnits-chatbot
```

2. **Install Dependencies**

```bash
npm install
```

3. **Run Development Server**

```bash
npm run dev
```
